# pongurses
A Simple game base on ncurses

**require : ncurses.h

For Ubuntu user : You can install ```libncurses5-dev``` packet.
```
$ sudo apt-get install libncurses5-dev
```

Compile with this command.
```
$ cc -o main main.c -lncurses
```

Run and enjoy. ;P
```
$ ./main
```
